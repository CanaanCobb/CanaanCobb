HTML Tags References

Basic tags

Tag Name Description
<DOCTYPE html> !DOCTYPE html This tag lets the browser know that the html being used is html5.
<html>/</html> HTML This is the root tag for the html document and is placed after the doc declaration tag and after the end of the html document.
<head>/</head> Head This tag holds the meta data for the page which includes browser title tag, and links such as stylesheets.
<body>/</body> Body This tag holds all the content of the html document that will show inside the browser page.
<p>/</p> Paragraph This tag is used to write content for the webpage that will appear on the web browser.
<h1></h1> Heading This tag is used to place headings within html document and wanges from <h1> to <h6>, <h1> being the largest font for headings and <h6> being the smallest font for headings.

Basic tags (extras)

Tag Name Description
<div>/</div> Div This tag stands for division and is used to divide the html document into sections for readability and structure.
<ol>/</ol> Order List This tag lets us use and numbered list for things we want to list out in a specific order, must be paired with a <li> tag.
<ul>/</ul> Unordered List This tag lets us use and bullet point list for things we want to list but in no specific order, must be paired with a <li> tag.
<li>/</li> List Items This tag is used when listing items in some sort of order commonly used when building an ordered and unordered list.
